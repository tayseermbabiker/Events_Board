## Notes

### What's NOT included (and why)
- **Eventbrite**: No public API, requires Apify ($39/mo)
- **LinkedIn Events**: No API, hard to scrape systematically
- **Facebook Events**: ToS issues, requires paid tools

### Deduplication Strategy
Many events appear on multiple sources (e.g., GITEX on DWTC + 10times + government sites). Use these fields to deduplicate:
1. `registration_url` (most reliable)
2. `title` + `start_date` combination
3. `venue` + `start_date` combination
